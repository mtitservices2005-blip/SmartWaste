# Backend futuro SmartWaste

> Datos demo · no producción

La Alpha no levanta servicios reales. Este directorio define módulos esperados para una API municipal futura:

- `municipal-operations`: KPIs, rutas del día, cumplimiento y retrasos.
- `assignments`: asignación camión-conductor-ruta.
- `supervisor-review`: verificación de rutas e incidencias.
- `driver-mobile`: inicio, progreso y cierre de rutas desde móvil.
- `audit`: bitácora por tenant.
