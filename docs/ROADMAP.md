# Roadmap

## Alpha
- Demo visual local.
- Flujos municipales, ciudadanos y Master Admin.
- Datos demo sin producción.

## Beta
- Autenticación por roles.
- Persistencia Supabase con RLS.
- Realtime para camiones e incidencias.
- Evidencias en storage seguro.

## Piloto
- Onboarding de un ayuntamiento.
- Carga de rutas reales aprobadas.
- Entrenamiento a supervisores y conductores.
- Métricas semanales de cumplimiento.
