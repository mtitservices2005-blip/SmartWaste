# Operación municipal Alpha

> Datos demo · no producción

El panel municipal concentra KPIs, rutas del día, camiones activos, cumplimiento, retrasos, incidencias, conductores, historial, filtros y buscador.

## Flujo demo

Ruta planificada → asignada → iniciada → en progreso → retrasada opcional → completada → verificada.

## Vistas

- Administrador: configuración y asignaciones.
- Supervisor: incidencias, retrasos y verificación.
- Conductor móvil: ruta asignada, avance y cierre.
